# Data

Сырые данные не коммитятся в репозиторий.

Датасет `QVHighlights` нужно скачать отдельно и положить локально.

Предполагаемая структура внутри `data/`:

- `raw/` — исходные файлы датасета
- `processed/` — подготовленные данные и промежуточные артефакты
- `embeddings/` — сохранённые визуальные и текстовые embedding-ы
