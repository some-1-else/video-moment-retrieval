# Project Brief

## Постановка задачи

Нужно исследовать задачу `video moment retrieval`: по текстовому запросу находить релевантный временной интервал в видео. Цель работы не в обучении новой SOTA-модели, а в сравнении реалистичных подходов по качеству и вычислительной стоимости.

## Что входит в scope

- обзор подходов к `text-to-video moment retrieval`
- основной датасет `QVHighlights`
- воспроизводимый `CLIP-based` baseline
- эксперименты с кадрами и temporal windows
- сравнение вариантов агрегации similarity score
- оценка качества и вычислительной стоимости inference

## Что не входит в scope

- обучение большой модели с нуля
- сложный distributed training pipeline
- масштабная инженерная инфраструктура
- широкий бенчмарк по множеству датасетов
- обязательная реализация специализированных моделей уровня `Moment-DETR` или `Lighthouse`

## Датасет

Основной датасет: `QVHighlights`.

Почему он подходит:

- содержит текстовые запросы и размеченные временные моменты
- хорошо соответствует задаче moment retrieval
- достаточен для исследовательского baseline без лишнего расширения scope

## Baseline

Базовый подход: `CLIP-based retrieval`.

Идея baseline:

- извлекать кадры или окна из видео
- считать визуальные embedding-ы
- кодировать текстовый запрос
- искать наиболее похожий кадр или временное окно

## Эксперименты

Планируемые эксперименты:

- поиск по отдельным кадрам
- поиск по `temporal windows`
- сравнение разных размеров окон
- сравнение способов агрегации embedding-ов или similarity
- сглаживание `similarity score`

Опционально:

- сравнение со специализированным подходом (`Moment-DETR` или `Lighthouse`), только если это реально успеем без риска для основного результата

## Метрики

Основные метрики:

- `Recall@1` при `IoU = 0.3`
- `Recall@1` при `IoU = 0.5`
- `Recall@1` при `IoU = 0.7`
- `mIoU`
- `inference time`
- число обработанных кадров
- размер сохранённых embedding-ов

## Expected Output

На выходе должен получиться:

- короткий и воспроизводимый pipeline для retrieval
- набор экспериментов с понятными настройками
- таблицы с качеством и вычислительными затратами
- выводы о том, какие упрощённые решения дают разумный компромисс между качеством и стоимостью
